## How to capture

1. Deploy Idium and restart HA once
2. Set theme to **idium_dark**
3. Hard-refresh the client
4. Capture at **2×** resolution for retina clarity
5. Crop to content; avoid HA sidebar if possible (or crop consistently)
6. Save as PNG in this directory
7. Update root `README.md` image links from `.svg` to `.png`
