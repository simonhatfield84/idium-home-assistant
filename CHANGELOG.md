# Changelog

## [1.0.0] - 2026-06-29

First public version — what's running at my house.

### Added

- `idium_dark` and `idium_light` themes
- Dashboard generator (`generator/idium_gen.py`) → 12 Lovelace storage files
- Home, room, Security, Climate, and Office dashboards
- Two-row nav + “More” section
- Example config (`config/idium.example.json`) and helpers package
- Docs, HACS metadata, CI that checks the generator compiles

### Notes

- Entity IDs are from my setup — you'll need to change them
- Deploying `.storage` files needs one HA restart
